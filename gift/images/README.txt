============================================
          WATCH IMAGES FOLDER
============================================

Add your watch images here!

Naming convention:
- watch1.jpg (or .png, .webp)
- watch2.jpg
- watch3.jpg
- etc.

Then update the WATCHES array in app.js to match your images.

Example entry in app.js:
{ id: 1, name: "Beautiful Gold Watch", image: "images/watch1.jpg" },

Tips:
- Use square or portrait images for best results
- Recommended size: 600x800 pixels or similar
- Supported formats: JPG, PNG, WebP
